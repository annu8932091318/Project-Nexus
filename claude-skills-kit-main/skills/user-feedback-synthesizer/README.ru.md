# user-feedback-synthesizer

Преврати папку с транскриптами интервью, тикетами поддержки и файлами опросов в структурированный инсайт-отчёт — без двух дней ручного тегирования и кластеризации.

---

## Что делает скил

`user-feedback-synthesizer` читает твои feedback-файлы (`.md`, `.txt`, `.csv`) и создаёт единый отчёт с:

- **Темами, ранжированными по частоте** — сколько источников упоминает каждую тему
- **Дословными цитатами** как доказательством по каждой теме
- **Таблицей приоритизированных инсайтов** (Высокий / Средний / Низкий по количеству источников)
- **Открытыми вопросами** — пробелы, противоречивые сигналы, единичные находки для дальнейшего изучения

Выходной файл `feedback-insights-YYYY-MM-DD.md` готов к передаче команде или вставке в плановый документ.

---

## Требования

- Один или несколько feedback-файлов в формате `.md`, `.txt` или `.csv`
- Файлы должны содержать транскрипты пользовательских интервью, экспорты тикетов поддержки, ответы на опросы или любой другой текстовый фидбек
- Файлы `.csv` должны иметь строку заголовков; колонка с контентом определяется автоматически

---

## Как использовать

1. Запусти скил одной из фраз ниже
2. Укажи путь к папке или список файлов
3. По желанию задай фокус: «фокус на болях при онбординге»
4. Claude обработает все файлы и сохранит `feedback-insights-YYYY-MM-DD.md` в твоей рабочей папке

---

## Примеры

**Пример 1 — Анализ папки с транскриптами интервью**

> «Синтезируй фидбек из папки interviews/»

Claude читает все `.md` и `.txt` файлы в `interviews/`, кластеризует темы и создаёт:
- 5 тем, ранжированных по частоте
- Лучшие цитаты по каждой теме
- Таблицу приоритизированных инсайтов
- Открытые вопросы на основе пробелов и противоречивых сигналов

**Пример 2 — CSV-опрос с исследовательским вопросом**

> «Отчёт по пользовательскому фидбеку из q1-survey.csv — фокус на опыте оформления заказа»

Claude читает `q1-survey.csv`, извлекает сигналы, связанные с оформлением заказа, выявляет темы и фиксирует инсайты и открытые вопросы по этому флоу.

## Результат

`feedback-insights-YYYY-MM-DD.md` с разделами:

- **Краткое резюме** — обзор из 3–5 предложений
- **Темы** — ранжированный список с типом сигнала, описанием и дословными цитатами
- **Приоритизированные инсайты** — таблица с Приоритет / Инсайт / Тема / Источники
- **Открытые вопросы** — пробелы в данных и единичные сигналы для дальнейшего изучения
- **Обработанные файлы** — список файлов с данными и файлов без сигналов
