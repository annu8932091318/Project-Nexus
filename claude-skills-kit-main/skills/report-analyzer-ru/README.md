# Report Analyzer — Claude Cowork Skill

Скилл для Claude Cowork, который анализирует большие отчёты (PDF, PPTX) и формирует структурированное резюме с ключевыми данными и инсайтами.

## Для кого

Продакт-менеджеры, маркетологи, аналитики, финансисты, топ-менеджмент — все, кто работает с большим потоком публичных отчётов от консалтинговых компаний и исследователей рынка.

## Что решает

- Экономит время на чтение отчётов в 50–100 страниц
- Позволяет быстро оценить содержание и ценность отчёта
- Извлекает ключевые цифры, данные и инсайты в единый компактный документ

## Что делает

1. Принимает файл отчёта (PDF или PPTX)
2. Задаёт 3 уточняющих вопроса: язык, фокус анализа, формат файла
3. Считывает и анализирует содержимое
4. Формирует структурированный документ (до 1,5 страниц)

## Структура выходного документа

- **Формат отчёта** — метаданные: название, автор, дата, объём, язык, тип
- **Executive Summary** — суть в 3-5 предложениях
- **Ключевые цифры и данные** — таблица с метриками и контекстом
- **Ключевые инсайты** — 5-7 пунктов
- **Структура отчёта** — оглавление с аннотациями к разделам

## Поддерживаемые форматы

**Входные:** PDF, PPTX (на русском и английском языках)

**Выходные:** .md, .pdf, .docx (на выбор пользователя)

## Триггеры

`проанализируй отчёт`, `разбери отчёт`, `summary отчёта`, `analyze report`, `report analysis`, `что в отчёте`, `краткое содержание отчёта`

## Требования

- Claude Cowork (десктопное приложение)

Все необходимые библиотеки устанавливаются автоматически при первом запуске скилла. Пользователю не нужно ничего устанавливать вручную.
