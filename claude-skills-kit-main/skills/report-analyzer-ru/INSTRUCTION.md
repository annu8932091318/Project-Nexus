# Инструкция: установка и использование Report Analyzer

## Скачивание с GitHub

### Вариант 1 — Скачать ZIP

1. Перейди на страницу репозитория на GitHub
2. Нажми **Code > Download ZIP**
3. Распакуй архив — внутри папка `report-analyzer/` с файлом `SKILL.md`

### Вариант 2 — Клонировать репозиторий

```bash
git clone https://github.com/YOUR_USERNAME/report-analyzer.git
```

### Вариант 3 — Скачать только SKILL.md

На странице файла `SKILL.md` в GitHub нажми **Raw**, затем сохрани страницу (Ctrl+S / Cmd+S).

---

## Установка в Claude Cowork

### Способ 1 — Через Settings (рекомендуемый)

1. Открой Claude Cowork
2. Перейди в **Settings > Customize > Skills**
3. Нажми **Add Skill**
4. Укажи путь к папке `report-analyzer/` на своём компьютере

### Способ 2 — Через команду

В окне Cowork:
```
/plugin add report-analyzer
```

### Способ 3 — Вручную

Скопируй папку `report-analyzer/` в директорию скиллов Cowork:
```
~/.claude/skills/report-analyzer/SKILL.md
```

После установки скилл активируется автоматически при использовании триггерных фраз.

## Использование

### Шаг 1 — Подготовь файл отчёта

Помести файл отчёта (PDF или PPTX) в рабочую папку проекта Cowork — например, в папку `input/`. Или убедись, что файл находится в любой доступной Cowork директории.

### Шаг 2 — Запусти скилл

Напиши в Cowork одну из триггерных фраз и укажи имя файла:

```
проанализируй отчёт McKinsey_Global_AI_2025.pdf
```

```
разбери отчёт input/CB-Insights-State-of-AI-Q4.pptx
```

```
analyze report Statista_EV_Market_Report.pdf
```

Claude найдёт файл в рабочей папке проекта и начнёт обработку.

### Шаг 3 — Ответь на 3 вопроса

Claude задаст интерактивные вопросы:
1. **Язык результата** — русский или английский
2. **Фокус анализа** — общий обзор / цифры и данные / стратегические выводы / всё вместе
3. **Формат файла** — .md / .pdf / .docx

### Шаг 4 — Получи результат

Claude сформирует файл с резюме и предложит скачать. Файл сохраняется в папку `output/` проекта.

---

## Структура результата

Выходной документ (до 1,5 страниц) содержит:

- **Формат отчёта** — метаданные: название, автор, дата, объём, язык, тип
- **Executive Summary** — суть отчёта в 3-5 предложениях
- **Ключевые цифры и данные** — таблица с метриками и контекстом
- **Ключевые инсайты** — 5-7 пунктов с конкретными выводами
- **Структура отчёта** — оглавление с аннотациями к разделам

## Частые вопросы

**Нужно ли устанавливать Python или библиотеки?**
Нет. Cowork устанавливает все необходимые зависимости автоматически при первом запуске скилла.

**Работает ли со сканированными PDF?**
Да. Если текст не извлекается напрямую, скилл автоматически запускает OCR.

**Можно ли анализировать отчёт на английском и получить результат на русском?**
Да. Язык входного файла и язык результата — независимые настройки.
