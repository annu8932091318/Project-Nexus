# Как установить memory-auditor-chat

Пошаговая инструкция по добавлению **memory-auditor-chat** в ваш Claude.ai.

---

## Что понадобится

- Аккаунт Claude (бесплатный или платный)
- Доступ к Claude.ai (веб-интерфейс)

> **Примечание:** Этот скилл предназначен для Claude.ai. Для Cowork используйте [memory-auditor-cowork](../../memory-auditor-cowork/).

---

## Вариант 1: Claude.ai (Projects) — Рекомендуемый

1. Откройте [claude.ai](https://claude.ai) и войдите в аккаунт
2. Нажмите **Projects** в левой панели
3. Создайте новый проект или откройте существующий
4. Нажмите на иконку **Project settings**
5. Найдите раздел **Custom instructions**
6. Скопируйте всё содержимое файла `SKILL.md` и вставьте в поле инструкций
7. Нажмите **Save**
8. Начните новый диалог внутри проекта — скилл активен

---

## Вариант 2: Claude.ai (Общий чат)

1. Откройте [claude.ai](https://claude.ai) и войдите в аккаунт
2. Начните новый диалог
3. Вставьте всё содержимое файла `SKILL.md` первым сообщением
4. Следом отправьте триггерную фразу (например, "аудит памяти")

> В режиме общего чата скилл проверяет только глобальные Memory Edits и Memory Summary. Проектные слои недоступны.

---

## Проверка установки

Начните новый диалог и попробуйте одну из фраз:

- "аудит памяти"
- "проверь память"
- "почисти память"

Если Claude ответит обзором памяти с указанием контекста и статистики — установка прошла успешно.

---

## Решение проблем

| Проблема | Решение |
|----------|---------|
| Скилл не срабатывает | Убедитесь, что скопировали **весь** файл SKILL.md, включая YAML-заголовок |
| Работает частично | Проверьте, что используете скилл в Claude.ai (веб-интерфейс), а не в Cowork |
| Неправильный язык | Скилл поддерживает русский и английский — попробуйте фразу на другом языке |
| Сообщение "инструмент недоступен" | Скилл автоматически переключится в режим только чтение; это ожидаемое поведение в некоторых средах |
