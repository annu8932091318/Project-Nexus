# Claude Memory Auditor (Chat) — Аудит и очистка памяти Claude

Находит противоречия, устаревшие записи, дубли и шум во всех слоях нативной памяти Claude.ai.

---

## Что делает

Memory Auditor Chat проверяет все слои нативной памяти Claude — Memory Edits, Memory Summary, Project Summary и Project Instructions — на проблемы, которые со временем снижают качество работы Claude. Скилл адаптирует объём проверки под контекст Claude.ai (общий чат или проект) и предлагает конкретные правки только с явным подтверждением пользователя.

> **Примечание:** Этот скилл предназначен для Claude.ai (веб-интерфейс). Для Cowork используйте [memory-auditor-cowork](../memory-auditor-cowork/).

## Как работает

1. **Проверка инструментов и контекста** — проверяет доступность инструментов, определяет среду и доступные слои памяти
2. **Обзор** — показывает статистику памяти и количество потенциальных проблем (можно остановиться на этом шаге)
3. **Глубокий анализ** — классифицирует проблемы по четырём категориям: противоречия, устаревшее, дубли, шум
4. **Действия** — для Memory Edits: прямое удаление/замена; для остальных слоёв: пошаговые инструкции для ручного исправления

## Быстрый старт

Откройте диалог с Claude и скажите:

- "аудит памяти"
- "проверь память"
- "почисти память"

Claude проведёт аудит и покажет результаты в структурированном отчёте, сгруппированном по приоритету.

## Категории проблем

- **Противоречие** — конфликт между записями в разных слоях или внутри одного слоя (высший приоритет)
- **Устаревшее** — факт изменился по данным более поздних записей или истории чатов (высший приоритет)
- **Дубль** — смысловое пересечение записей, которые можно объединить (средний приоритет)
- **Шум** — корректные, но малоценные записи: промежуточные состояния, разовые вопросы, избыточные формулировки (низкий приоритет)

## Поддерживаемые среды

| Среда | Доступные слои | Авто-исправление |
|-------|---------------|-----------------|
| Общий чат Claude.ai | Memory Edits, Memory Summary | Memory Edits |
| Проект Claude.ai | Memory Edits, Memory Summary, Project Summary, Project Instructions | Memory Edits |

## Требования

- Аккаунт Claude (бесплатный или платный)
- Доступ к Claude.ai (веб-интерфейс)
- Внешние инструменты и API-ключи не требуются

## Безопасность

Memory Auditor никогда не выполняет полный сброс памяти. Каждое изменение Memory Edits требует явного подтверждения. Скилл не может напрямую изменять Memory Summary, Project Summary и Project Instructions — вместо этого предоставляет пошаговые инструкции для ручного исправления.
