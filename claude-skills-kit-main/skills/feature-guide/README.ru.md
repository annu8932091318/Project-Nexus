# Feature Guide — Карточки возможностей Claude по запросу

Узнайте, что умеет Claude, без долгих поисков в документации.

---

## Что делает скил

Claude Feature Scout превращает вопросы вида «есть ли у Claude память?» или «как использовать Projects?» в структурированную карточку фичи — без необходимости читать документацию.

Каждая карточка содержит:
- Что делает фича (простым языком, без терминологии)
- Где доступна (Claude.ai, Projects, Claude Code, мобильное приложение и др.)
- Какой тариф нужен (Free, Pro, Team, Enterprise)
- Как включить или использовать
- Актуальные ограничения
- Текущий статус (доступна, бета, только Pro, анонсирована)
- Вердикт: подходит ли фича для вашей конкретной задачи

Скил выполняет живой поиск по официальной документации Anthropic — статус фичи всегда актуален.

---

## Как работает

1. Вы описываете фичу по названию или формулируете задачу
2. Скил ищет актуальный статус в документации Anthropic
3. Возвращает структурированную карточку — готовую к действию
4. При необходимости — исследуете смежные фичи в той же сессии

---

## Быстрый старт

Откройте диалог с Claude (с установленным скилом) и напишите:

- «Есть ли у Claude память?»
- «Что такое Projects?»
- «Хочу чтобы Claude помнил мой контекст между сессиями»
- «Поддерживает ли Claude голос?»
- «Умеет ли Claude создавать и сохранять файлы?»

---

## Пример

**Запрос:** «Есть ли у Claude память?»

**Ответ:**
```
## Memory (Projects)

Что это: Claude сохраняет контекст между разговорами внутри Projects.
  Каждый проект имеет свою независимую память.

Где доступно: Claude.ai — тарифы Free и Pro

Как использовать: Создайте Project на claude.ai, ведите диалоги внутри него.
  Claude будет помнить контекст предыдущих сессий в рамках проекта.

Ограничения:
- Память ограничена одним проектом, не работает между проектами
- Недоступна в обычных диалогах вне Projects

Статус: Доступна

Вердикт: Подходит — если нужна память между сессиями, Projects — правильная фича.

Исследовать смежные фичи: Custom Instructions, Knowledge base?
```

---

## Требования

- Аккаунт Claude (бесплатный или платный)
- Web search должен быть доступен в сессии Claude для получения актуального статуса

Техническая настройка не требуется. Работает в Claude.ai, Claude Projects, Claude Cowork и Claude Code.

---

## Когда использовать

- Перед тем как тратить время на поиск в документации Anthropic
- Когда не уверены, умеет ли Claude что-то конкретное
- Когда статус фичи мог измениться (бета → доступна и т.п.)
- При знакомстве нового пользователя с возможностями Claude
