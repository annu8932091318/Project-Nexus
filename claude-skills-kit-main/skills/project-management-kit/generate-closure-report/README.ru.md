# generate-closure-report

Генерирует отчёт о закрытии проекта — итоговый артефакт жизненного цикла проекта.

## Описание

Агрегирует все данные проекта в итоговый документ: результаты vs цели, анализ отклонений, реализованные риски, извлечённые уроки, заметки по передаче. Последний документ, создаваемый агентом.

**Фаза:** 6 — Закрытие
**Результат:** `closure-report.md`

## Триггеры

| Язык | Команды |
|------|---------|
| Русский | «сформируй отчёт о закрытии», «закрой проект», «подготовь closure report», «отчёт о закрытии проекта» |
| English | "generate closure report", "close project", "project closure report", "prepare closure report" |

## Входные данные

| Данные | Обязательно | Примечание |
|--------|:-----------:|------------|
| Устав проекта | да | Цель, результаты, команда, спонсор, бюджет, сроки |
| Отчёт план/факт | да | Сводка отклонений, фактические даты/бюджет/скоуп |
| Извлечённые уроки | да | В чате: результаты ретроспективы, обратная связь команды, наблюдения PM |
| Реестр рисков | нет | Для анализа реализованных рисков |
| Открытые вопросы | нет | Задачи/вопросы, оставшиеся после закрытия |

## Структура результата

Отчёт содержит: резюме проекта (из устава), оценку достижения целей, анализ отклонений (из отчёта план/факт), реализованные риски и их влияние, извлечённые уроки (по категориям), открытые вопросы и заметки по передаче, блок финального согласования.

## Зависимости

**Требует:** `generate-charter` + `generate-plan-fact-report` (оба утверждены) + извлечённые уроки от пользователя

**Разблокирует:** нет зависимых скиллов. Финальный скилл жизненного цикла проекта.

## Пример

> Сформируй отчёт о закрытии. Устав и отчёт план/факт в знаниях. Извлечённые уроки: недооценили QA на 30%, надо было раньше начинать интеграционное тестирование; коммуникация с заказчиком хорошо работала через еженедельные демо; риск R003 (задержка подрядчика) реализовался, но уложились в буфер.
