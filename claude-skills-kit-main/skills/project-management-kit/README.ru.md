# Набор навыков для управления проектами в Claude

Набор навыков, которые автоматизирует создание проектной документации для команд разработки IT-продуктов. Реализован как набор скиллов для Claude.ai Projects.

## Что делает

Агент генерирует структурированные проектные документы через диалог. Вы предоставляете данные о проекте — агент задаёт уточняющие вопросы, применяет методологию PMBoK 8 и Agile, и выдаёт готовые документы.

**MVP включает 7 скиллов и 8 задач проектного управления:**

| Документ | Фаза проекта | Описание |
|----------|--------------|----------|
| Устав проекта | Инициация | Цели, скоуп, сроки, бюджет, команда, ограничения |
| Реестр рисков (первичный) | Инициация | Идентификация рисков с оценкой вероятности/влияния и стратегиями реагирования |
| План проекта | Планирование | WBS, вехи, сроки, распределение ресурсов |
| План коммуникаций | Планирование | Матрица стейкхолдеров, расписание коммуникаций, правила эскалации |
| Реестр рисков (детальный) | Планирование | Обогащение первичного реестра рисками из плана и уточнёнными оценками |
| Протокол встречи | Любая фаза | Структурирование заметок со встречи: решения, задачи, изменения плана |
| Отчёт план/факт | Закрытие | Сравнение плановых и фактических сроков, бюджета, результатов |
| Отчёт о закрытии | Закрытие | Итоги проекта, отклонения, реализованные риски, извлечённые уроки |

## Для кого

Менеджеры проектов, CPO, продуктовые лиды, тимлиды — все, кто управляет проектами по разработке или запуску IT-продуктов и хочет сократить время на рутинную документацию.

Техническая подготовка не требуется. Агент работает через естественный диалог в Claude.ai.

## Как работает

1. **Начните сессию** — укажите агенту фазу проекта и текущую задачу.
2. **Предоставьте данные** — бриф, ответы заказчика, ограничения или любые неструктурированные заметки.
3. **Агент генерирует документ** — следуя алгоритму скилла и шаблону.
4. **Проверьте и утвердите** — запросите правки или подтвердите результат.
5. **Переходите к следующей задаче** — агент подскажет, какие задачи стали доступны.

Скиллы связаны цепочкой: результат одного скилла становится входом для следующего. Например, утверждённый устав разблокирует генерацию реестра рисков и плана проекта.

## Методология

Агент применяет гибрид PMBoK 8 (структура процессов, типы артефактов, управление рисками) и Agile (итеративная поставка, лёгкая документация, адаптивное планирование). Фокус — проекты по разработке и запуску IT-продуктов.

## Поддержка языков

Агент автоматически определяет язык запроса:

- Запрос **на русском** → документы и ответы на русском
- Запрос **на английском** → документы и ответы на английском

Все шаблоны двуязычные (RU/EN).

- Подписка Claude Pro или Max
- Функция Projects в Claude.ai
