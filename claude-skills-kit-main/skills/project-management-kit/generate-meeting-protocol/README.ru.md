# generate-meeting-protocol

Генерирует структурированный протокол встречи из свободных заметок.

## Описание

Превращает неструктурированные заметки со встречи в формальный протокол с решениями, задачами и предложениями по изменению проектных документов. Используется на любой фазе проекта.

**Фаза:** любая (независимый скилл)
**Результат:** `meeting-protocol-{date}.md`

## Триггеры

| Язык | Команды |
|------|---------|
| Русский | «оформи протокол», «протокол встречи», «обработай заметки со встречи» |
| English | "generate meeting protocol", "meeting notes", "meeting minutes" |

## Входные данные

| Данные | Обязательно | Примечание |
|--------|:-----------:|------------|
| Заметки со встречи | да | Любой формат: тезисы, свободный текст, копия чата, расшифровка диктофона |
| План проекта | нет | Если есть — изменения привязываются к рабочим пакетам (WP-xxx) |
| Предыдущие протоколы | нет | Для отслеживания выполнения задач |

## Структура результата

Протокол содержит: метаданные встречи (дата, участники, повестка), ключевые обсуждения, принятые решения, задачи (кто, что, когда), предложения по изменению плана или других документов.

## Зависимости

**Требует:** заметки со встречи от пользователя (предыдущие скиллы не нужны).

**Разблокирует:** нет зависимых скиллов. Автономный утилитарный скилл.

## Пример

> Вот заметки с сегодняшнего демо спринта. Участники: PM, 2 разработчика, дизайнер, заказчик. Обсудили прототип, заказчик запросил 2 изменения, решили сдвинуть запуск на 1 неделю.

Агент создаёт структурированный протокол с решениями, задачами с владельцами и дедлайнами, и предложением по обновлению плана проекта.
