# one-to-one-prep — Генератор prep-документа для встреч 1-on-1

> Превратите заметки и данные таск-трекера в структурированную агенду за секунды.

**one-to-one-prep** — скил для Claude, который помогает тимлидам, Engineering Manager'ам и руководителям отделов готовиться к ежемесячным встречам 1-on-1. Вставьте заметки с прошлой встречи и список задач сотрудника — получите готовый prep-документ со статусом action items, приоритизированными темами для обсуждения и вопросами по мотивации.

Никаких интеграций. Работает с обычным текстом из любого таск-трекера.

---

## Что делает

- Отслеживает action items с прошлого 1-on-1: Выполнено / В процессе / Просрочено
- Анализирует данные таск-трекера: завершённые задачи, застрявшие, блокеры, аномалии
- Генерирует 3–5 приоритизированных тем для обсуждения на основе реального контекста
- Предлагает открытые вопросы по мотивации и wellbeing, релевантные ситуации
- Выводит готовый шаблон для фиксации новых action items прямо на встрече
- При первой встрече (нет истории) — автоматически переключается в режим онбординга

---

## Как работает

1. Вставьте заметки с прошлого 1-on-1 (или укажите, что это первая встреча)
2. Вставьте список задач сотрудника из любого трекера (Jira, Linear, Asana, ClickUp, Notion или обычный текст)
3. При желании добавьте карьерные цели, IDP или OKR — скил добавит вопросы на развитие
4. Получите структурированный prep-документ в Markdown, готовый к встрече

## Структура выходного документа

```
# 1-on-1 Prep: [Имя] — [дата]

## Action Items с прошлой встречи
| Задача | Статус | Комментарий |

## Ключевые события периода
Завершено / Под риском / Блокеры

## Темы для обсуждения
1. [Высокий] ...
2. [Средний] ...
3. [Низкий] ...

## Вопросы по мотивации и wellbeing
- ...

## Новые Action Items
| Задача | Ответственный | Срок |
```
